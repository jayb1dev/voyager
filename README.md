<h1 align="center"/>Seevoy</h1>

Seevoy helps you to see Lemmy with
enhanced large text support.

Seevoy is a fork of [Voyager](https://github.com/aeharding/voyager).

# Build for iOS on Mac

```
./build-ios.sh
```
